Overview

Noter is a minimalistic todo list app developed in Angular using the latest framework's update: Angular Signals.


The Design System includes design tokens following Material Design's visual style to provide clear structure of colors roles.


The goal was to make Noter an app that enhances the note-taking experience as easy and pleasant.
